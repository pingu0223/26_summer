## 题目描述
```text
给你一个非负整数数组 nums 和一个整数 target 。
向数组中的每个整数前添加 '+' 或 '-' ，然后串联起所有整数，可以构造一个 表达式 ：
例如，nums = [2, 1] ，可以在 2 之前添加 '+' ，在 1 之前添加 '-' ，然后串联起来得到表达式 "+2-1" 。
返回可以通过上述方法构造的、运算结果等于 target 的不同 表达式 的数目。
```
## 坑点解析
### 从求max到求方案数——递推公式与初始化的变化
初始化时最好先把i==0的情况初始化掉。
### weight=0的特殊情况
1. 若weight！=0的话，当j==0时只有一种情况，即什么物品都不取
2. 若weight==0的话，当j==0时，则是当前所有为w==0的数取组合，大概就是(int)pow(2,num_zero) 所以我们要记录num_zero
### target为负时
因为target的正负其实就是正数集合与负数集合调换时，所以可以直接把target变成正的target~
